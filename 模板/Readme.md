```diff
- 二叉搜索树(BST):左子树节点的值<根节点的值，右子树节点的值>=根节点的值

+ 中序遍历结果有序（不下降的顺序，有些临点可能相等）
- 如果二叉树的中序遍历不是不下降序列，则一定不是BST
- 如果二叉树的中序遍历是不下降，也未必是BST
+ BST的高度：最好O(n);最坏o(logN);用o(h)表示更合适
```

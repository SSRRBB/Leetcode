## 答案：
**前缀和**
```python
class Solution:
    def numberOfSubarrays(self, nums: List[int], k: int) -> int:
        ## 同560***
        res = 0
        odd_num = collections.Counter()
        odd_num[0] = 1
        cur_odd_num = 0
        for i in nums:
            if i % 2 != 0:
                cur_odd_num += 1
            odd_num[cur_odd_num] += 1
            
            if cur_odd_num - k in odd_num:
                res += odd_num[cur_odd_num - k]
        return res

```
